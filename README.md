## Overview
Tomodachi PC is a complete solution for playing Tomodachi Life on PC. It includes optimized emulator configurations (Citra, Ryujinx, Yuzu / Eden), Mii maker tools, QR code generator, save editor, custom mods, English patch, RomFS modifications, texture packs, and 60 FPS enhancements.
